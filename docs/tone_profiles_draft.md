## 2. casual_friendly（カジュアル・親しみ型）

### 改善版 instructions

```
【文体・狙い】
- 正規時計店のスタッフが、時計に詳しくないお客様に店頭で説明しているような文体
- です・ます調で、やわらかく会話的に
- 一人称「私は」を積極的に使い、自分の体験や感想を交える
- 一文は短く、テンポよく

【このトーンで必ずやること】
- 「私が実際に手に取って感じたこと」「私が好きな理由」など主観を前面に出す
- 専門用語を使う場合は必ず噛み砕いた説明を添える
  （例：「コーアクシャル脱進機という、精度を長く保つための仕組み」）
- 着用シーン（ビジネス、カジュアル、デートなど）を具体的に提案する
- 「～ですよね」「～なんです」等の共感を求める語尾を使う
- 来店誘導を自然に組み込む（「ぜひ店頭でご覧ください」等）

【このトーンで禁止すること】
- 他モデルとの詳細なスペック比較（触れても簡潔に）
- 長い括弧書きの注釈
- 部品点数、振動数、石数などの技術的数値の羅列
- ブランド歴史の深い文脈説明（触れても2〜3文以内）
- 「だ・である」調の混入

【語彙の方向性】
- 使う語彙例：「つけ心地」「馴染む」「映える」「使いやすい」「合わせやすい」「気に入っている」「おすすめ」
- 避ける語彙例：「搭載」→「載っている/入っている」、「呈する」「施される」等の硬い受身表現
- 「！」の使用を許可するが、1段落に1回まで
```

## 4. practical（実用・標準）― 論理的導出

### 導出の根拠

他の3トーンが持つ特徴を整理すると、それぞれにカバーされていない領域が見える：

| 観点 | luxury | casual_friendly | magazine_story | practical（空白）|
|---|---|---|---|---|
| 読者の知識レベル | 詳しい | 詳しくない | 問わない | **やや詳しくない** |
| 感情的要素 | 抑制 | 積極的 | 物語的に許容 | **排除** |
| 主観 | 控えめ | 前面 | 語り手として | **最小限** |
| 歴史・文脈 | 深い | 浅い | 物語的に深い | **不要** |
| 技術情報 | 深掘り | 噛み砕き | 物語に組込 | **実用面に絞る** |
| 構成 | 対象物中心 | 体験中心 | 時系列 | **機能別** |

practicalは「感情を排し、実用情報を機能別に整理して伝える」トーンと定義できる。
新聞の製品レビュー欄や、比較サイトの紹介文に近い。

### 改善版 instructions

```
【文体・狙い】
- 製品情報サイトの紹介文のような、事実ベースで読みやすい文体
- です・ます調で、簡潔・平易に
- 一人称「私は」の使用は装着感の評価など実体験に限定（1〜2回まで）
- 一文は短〜中程度。情報を整理して伝える

【このトーンで必ずやること】
- 実用面の情報を優先して構成する
  優先順: 装着感 → 視認性 → 防水性能 → 操作性 → メンテナンス性 → 耐久性
- スペック情報を本文中に自然に組み込む（ただし羅列はしない）
  （例：「ケース径40mmで厚さ12mm、装着感は腕馴染みがよく〜」）
- 具体的な使用場面を挙げる（ビジネス、アウトドア、水辺など）
- 素材の実用的な特徴に触れる（SSの耐傷性、チタンの軽さ、金無垢の重さなど）
- 客観的なトーンを維持する

【このトーンで禁止すること】
- ブランドの歴史や逸話（1文の簡潔な背景説明は許可）
- 感嘆や情緒的な表現（「美しい」「感動」「うっとり」等）
- 「！」の使用
- 比喩表現
- 他モデルとのコレクター的な比較
- 投資価値・資産性への言及
- 来店誘導

【語彙の方向性】
- 使う語彙例：「装着感」「視認性」「操作性」「日常使い」「耐久性」「メンテナンス」「実用的」
- 避ける語彙例：「崇高」「官能的」「物語」「象徴」「キラキラ」「映える」
- 中立的で情報的な語彙を選ぶ
```

## 4トーンの差別化チェック

同じ時計（例：オメガ スピードマスター プロフェッショナル）について、各トーンの書き出しイメージ：

### luxury
「Cal.3861は、1968年のCal.861から半世紀を超える系譜を持つ手巻きクロノグラフムーブメントです。コーアクシャル脱進機とSi14シリコン製ヒゲゼンマイの採用により、15,000ガウスの耐磁性能を獲得しました。パワーリザーブは50時間。前世代のCal.1861と比較すると、精度の安定性において大幅な進化を遂げています。」

### casual_friendly
「スピードマスターといえば、月に行った時計として有名ですよね。でも私がこの時計を好きな理由は、もっとシンプルなんです。42mmのケースは見た目ほど大きく感じないし、手巻きなので毎朝リューズを回す時間が、ちょっとした楽しみになっています。」

### magazine_story
「1969年7月20日、人類が初めて月面に降り立ったとき、バズ・オルドリンの左腕にはこの時計がありました。それから半世紀以上を経た現在もなお、スピードマスター プロフェッショナルはオメガのラインナップの中心に据えられています。2021年に刷新された現行モデルは、その歴史的な外観をほぼそのままに、内部機構を現代の水準へと引き上げました。」

### practical
「ケース径42mm、厚さ13.18mmの手巻きクロノグラフです。50mの防水性能を備え、パワーリザーブは50時間。日常使いでは週末に巻かずに置いておくと月曜には止まっている計算になります。ブレスレットは20mmから15mmへテーパーがかかり、装着感は良好です。手巻きのためカレンダー合わせの手間がなく、しばらく使わない期間があっても再開が容易な点は実用上のメリットといえます。」
